## 검사 기준

### G1: 렌더링 (필수)
- [ ] 페이지 정상 로드 (HTTP 200, no JS error)
- [ ] 핵심 헤딩/텍스트 노출
- [ ] 콘솔 에러 0건

### G2: BE 연동 (필수)
- [ ] API 호출 성공 (no 500/401/404)
- [ ] 실제 DB 데이터 노출 (USE_MOCK=false)
- [ ] 로딩 상태 → 데이터 표시 완료

### G3: 인증/인가 (P0)
- [ ] 미인증 → 로그인 리디렉트
- [ ] 인증 후 정상 접근
- [ ] 로그아웃 동작

### G4: 핵심 상호작용 (P0~P1별)
- [ ] CRUD 기본 동작
- [ ] 폼 유효성 검사

### G5: 성능/UX
- [ ] LCP < 3s
- [ ] 모바일 반응형
