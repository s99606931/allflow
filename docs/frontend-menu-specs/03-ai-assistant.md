# AI 어시스턴트 패널 (AI Assistant Panel)

> 파일: `src/components/shell/ai-panel.tsx`, `src/components/ai/`

## 개요

우측 슬라이드인 패널로 표시되는 워크스페이스 AI 채팅 인터페이스.  
Claude API와 연동하여 스트리밍 응답, RAG, 툴 호출, 멀티모달 첨부를 지원한다.

## 기능 목록

### 1. 패널 컨트롤

| 기능 | 설명 | 구현 | 백엔드 | 테스트 |
|------|------|------|--------|--------|
| 패널 열기/닫기 | 탑바 Sparkles 버튼 또는 ⌘I 단축키 | ✅ | 🔌 (zustand) | ⬜ |
| 패널 너비 조절 | 좌측 엣지 드래그로 너비 조절 | ✅ | 🔌 (zustand persist) | 🧪 use-resize-drag.test.tsx |
| 대화 초기화 | Trash2 아이콘 버튼 → 현재 대화 삭제 | ✅ | 🔗 DELETE /ai/threads/:id | 🧪 use-ai.test.tsx |
| 스레드 사이드바 토글 | PanelLeft 버튼 → 대화 히스토리 사이드바 | ✅ | 🔌 | ⬜ |

### 2. AI 채팅 인터페이스

| 기능 | 설명 | 구현 | 백엔드 | 테스트 |
|------|------|------|--------|--------|
| 메시지 전송 | Enter로 전송, Shift+Enter로 줄바꿈 | ✅ | 🔗 POST /ai/chat (SSE 스트리밍) | 🧪 use-ai.test.tsx |
| 스트리밍 응답 | SSE로 토큰 단위 실시간 출력 | ✅ | 🔗 | 🧪 |
| 마크다운 렌더링 | AI 응답을 마크다운으로 파싱·렌더링 | ✅ | 🔌 (클라이언트 파싱) | ⬜ |
| 인용 뱃지 | 응답에 사용된 문서 출처 표시 | ✅ | 🔗 (RAG 결과 메타) | ⬜ |
| 툴 호출 트레이스 | AI가 호출한 도구 목록 펼침/접힘 표시 | ✅ | 🔗 (tool_call 메타) | 🧪 |
| 토큰 사용량 표시 | prompt↑ + completion↓ = 합계 · 비용 · 모델 | ✅ | 🔗 (usage 메타) | ⬜ |
| 추천 질문 칩 | 빈 대화 시 제안 질문 버튼 4개 표시 | ✅ | 🔌 (하드코딩) | ⬜ |
| 로딩 상태 | "생각 중…" 스피너 표시 | ✅ | 🔌 | ⬜ |

### 3. AI 컴포저 (입력창)

| 기능 | 설명 | 구현 | 백엔드 | 테스트 |
|------|------|------|--------|--------|

| 파일 첨부 | 이미지/파일 첨부 → presigned URL 업로드 | ✅ | 🔗 POST /ai/attachments | 🧪 use-file-attach.test.tsx |
| 음성 입력 | 마이크 버튼 → Web Speech API → 텍스트 변환 | ✅ | 🔌 (브라우저 API) | 🧪 use-voice-input.test.tsx |
| 연결 상태 표시 | "온라인" / "기본 응답 모드" 상태 레이블 | ✅ | 🔗 (LLM 연결 상태) | ⬜ |

| 기능 | 설명 | 구현 | 백엔드 | 테스트 |
