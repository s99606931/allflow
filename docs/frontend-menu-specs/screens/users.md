# 사용자 관리 (Users)

> 경로: `/users` | 파일: `src/components/screens/users.tsx`  
> E2E: `tests/e2e/menus/org-users-hr.spec.ts`

## 개요

워크스페이스 사용자 디렉토리 및 관리. 사용자 조회·검색·초대·역할 변경을 지원한다.
