# 상단 탑바 (Top Bar)

> 파일: `src/components/shell/topbar.tsx`

## 개요

화면 상단에 고정된 글로벌 헤더. AI 패널 토글, 알림, 도움말, 프로필 메뉴를 제공한다.

## 기능 목록

### 1. 실시간 연결 상태 표시

| 기능 | 설명 | 구현 | 백엔드 | 테스트 |
|------|------|------|--------|--------|
| 연결 상태 닷 | 녹색(연결)/황색(지연)/회색(오프라인) 색상 닷 + 레이블 | ✅ | 🔗 WebSocket /realtime | 🧪 use-realtime.test.tsx |
| 상태 레이블 | "온라인" / "재연결 중" / "오프라인" 텍스트 표시 | ✅ | 🔗 | 🧪 |

### 2. AI 어시스턴트 토글 버튼

| 기능 | 설명 | 구현 | 백엔드 | 테스트 |
|------|------|------|--------|--------|
| AI 패널 열기/닫기 | Sparkles 아이콘 버튼 → 우측 AI 패널 토글 | ✅ | 🔌 (zustand) | ⬜ |
| 활성 상태 강조 | 패널 열림 시 버튼 accent 색상으로 표시 | ✅ | 🔌 | ⬜ |

### 3. 알림 메뉴

| 기능 | 설명 | 구현 | 백엔드 | 테스트 |
|------|------|------|--------|--------|
| 알림 뱃지 | Bell 아이콘에 읽지 않은 알림 수 표시 | ✅ | 🔗 GET /nav/counts | 🧪 nav.test.ts |
| 알림 드롭다운 | 최근 알림 목록 팝오버 표시 | ✅ | 🔗 GET /notifications | ⬜ |
| 알림 읽음 처리 | 드롭다운에서 개별 알림 읽음 처리 | ✅ | 🔗 PATCH /notifications/:id | ⬜ |
| 전체 읽음 처리 | "전체 읽음" 버튼 → 모든 알림 읽음 | ✅ | 🔗 PATCH /notifications/read-all | ⬜ |
| 알림 센터 링크 | "전체 보기" → `/notifications` 이동 | ✅ | 🔌 | ⬜ |

### 4. 도움말 메뉴

| 기능 | 설명 | 구현 | 백엔드 | 테스트 |
|------|------|------|--------|--------|
| 키보드 단축키 표시 | ⌘K, ⌘/, ⌘B, ⌘I 단축키 힌트 | ✅ | 🔌 | ⬜ |
| 문서 링크 | 외부 문서 URL 링크 | ✅ | 🔌 | ⬜ |
| 피드백 전송 링크 | 피드백 폼 URL 링크 | ✅ | 🔌 | ⬜ |
| 앱 버전 표시 | 현재 앱 버전 표시 | ✅ | 🔌 | ⬜ |

### 5. 사용자(프로필) 메뉴

| 기능 | 설명 | 구현 | 백엔드 | 테스트 |
|------|------|------|--------|--------|
| 프로필 정보 표시 | 이름·역할·부서·이메일 표시 | ✅ | 🔗 GET /auth/me | 🧪 use-data.test.tsx |
| 계정 설정 링크 | `/settings` 이동 | ✅ | 🔌 | ⬜ |
| 로그아웃 | 세션 종료 후 로그인 페이지 이동 | ✅ | 🔗 POST /auth/logout | ⬜ |

## 미구현 / 개선 필요 항목

| 항목 | 설명 | 우선순위 |
|------|------|----------|
| 알림 클릭 시 해당 화면 딥링크 | 알림 항목 클릭 → 관련 태스크/이슈로 이동 | 중 |
| 알림 드롭다운 E2E 테스트 | Playwright 커버리지 없음 | 중 |
| 글로벌 검색 바 (탑바) | 사이드바 검색과 중복 여부 검토 필요 | 낮음 |
